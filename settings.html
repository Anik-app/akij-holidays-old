⚙️ Settings — Cloud & Configuration

Set up automatic Google-Sheet logging and GitHub cloud storage. Everything you save from any maker page will sync to both places automatically.

📊 Google Sheets — Automatic Logging Not configured

When enabled, every document you save is automatically appended to your Google Sheet — with per-type tabs (Invoices, Vouchers, Airline Tickets, Ticket Invoices). Duplicate document IDs are updated in place.

📄 Open Target Google Sheet ↗

  1. Open your Google Sheet

    Click Open Target Google Sheet above (it opens Akij Holidays in a new tab). Sign in with your Google account if prompted.

  2. Open the Apps Script editor

    In the sheet, go to Extensions → Apps Script. A new editor tab opens.

  3. Paste the logger script

    Delete anything in Code.gs, click Copy Apps Script code above, paste it, and press Ctrl/Cmd + S to save. Name the project any name you like.

    
        
    4. 
        
  4. 
          
    Deploy as a Web App
    
          
    In the Apps Script editor, click Deploy → New deployment. Set:
    
          
    • Type: Web app  · 
             Execute as: Me  · 
             Who has access: Anyone
    
          
    Click Deploy, authorize when prompted, and copy the resulting URL
             (looks like https://script.google.com/macros/s/AKfyc…/exec).
    
        
    5. 
        
  5. 
          
    Paste the URL into config.js
    
          
    Open config.js in your project folder and paste the URL:
    
          
    APPS_SCRIPT_URL: "https://script.google.com/macros/s/AKfyc…/exec",
    
          
    Refresh this page. The badge above should turn ✓ Connected.
